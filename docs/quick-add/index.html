File format & field guide

Upload a .csv or .json file, or paste JSON below. Each row/object is one task. Only title is required — everything else has a default.

FieldAcceptsDefault
titletext (required)—
categorywork, school, personalpersonal
priority1/2/3 or high/med/low2 (med)
due_dateYYYY-MM-DDnone
due_timeHH:MM (24h)none
notestextnone
quick_printtrue/falsefalse

JSON — an array of objects:

[
  { "title": "Submit expense report", "category": "work", "priority": "high", "due_date": "2026-07-05" },
  { "title": "Renew license", "category": "work", "priority": 2, "due_time": "14:30", "quick_print": true },
  { "title": "Call vendor", "notes": "Ask about invoice #204" }
]

CSV — first row is the header (any subset/order of the fields above):

title,category,priority,due_date,due_time,notes,quick_print
Submit expense report,work,high,2026-07-05,,,false
Renew license,work,2,,14:30,,true
Call vendor,work,low,,,Ask about invoice #204,false
